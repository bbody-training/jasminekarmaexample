## How to run tests

### Headless (Chrome & Firefox)
`karma start karma.conf.js`
or
`npm test`

### Non-headless (Chrome, Firefox & Safari)
`karma start --browsers='Chrome,Firefox,Safari'`
or
`npm run test1`
